- [o] Vite 기반 Vue 3 (JavaScript) 프로젝트 생성
- [o] 라우터 (Vue Router) 및 상태 관리 (Pinia) 설정

- [o] 글로벌 디자인 토큰 (CSS 변수) 및 1:1 이미지 최적화 기본 스타일 (reset) 셋업
- [o] 공통 Layout 구성 (상단 헤더 고정, 반응형 분기)
